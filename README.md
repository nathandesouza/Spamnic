# Spamnic

Spamnic é um jogo plataforma 3d de ação em terceira pessoa ambientado em um mundo virtual, você controla um robô que está sendo treinado para acabar com o crime cibernético. O jogador utiliza mecânicas aprendidas para enfrentar Inimigos e quebra-cabeças para avançar nos níveis. A dificuldade dos níveis aumenta de acordo com o avanço
